# QA and project control logs

This folder contains the public QA and control documents that support the WERS workforce reward and satisfaction analysis.

The methodology log at the project root remains the main source of truth for methodology decisions. See `methodology_log.md`, Section 9, for the full documentation system.

## Public QA files

- `qa_validation_log.xlsx`  
  Cross-tool and cross-method validation checks, including SQL validation of weighted R outputs.

- `original_vs_revisit_log.xlsx`  
  Summary of deliberate departures from the original MSc analysis and rationale for the revised senior-analyst approach.

## Local operational file

- `daily_work_log.xlsx`  
  Private working log used to track Did / Learned / Next by session. This is kept locally and is not committed to GitHub.
