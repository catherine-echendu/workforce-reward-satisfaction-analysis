# Power BI

Stakeholder-facing dashboard. The .pbix file is treated as a working
source file (similar to code), not an output. Final exported visuals
or PDFs live in 04_outputs/figures/ if needed for the Word report.

Planned dashboard structure:
1. Overview        (headline satisfaction, three-band split, caveats)
2. Pay             (pay satisfaction gradient, grouped pay bands)
3. Benefits        (the locked benefit levers)
4. QA & method     (row counts, weights note, missingness, definitions)
